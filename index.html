Warning System

A warning system for the Swedish Red Cross made to detect certain hashtags in the live Twitter feed. If a hashtag is spotted by the system it will automatically send out a mail. This was created with Python, Twitter API and Gmail API.

Lunch tag system

A security system of sorts for NTI Gymnasiet Uppsala's lunch line. By matching the students tag id with the ones in the system we can detect if they're supposed to eat lunch, if it's their time to eat or if they have scanned and eaten already.
